# 01_小红书发布工作流

## 目标

从 Feishu 发布池读取内容，通过 browser-worker 模拟人工浏览器操作完成小红书笔记发布。

## 基本链路

Feishu 发布池
→ n8n 读取待发布记录
→ n8n 生成 publish_job
→ browser-worker 接收任务
→ 打开指定浏览器配置
→ 进入小红书发布页面
→ 上传图片
→ 填写标题、正文、标签
→ 点击发布
→ 截图留证
→ 回写 PostgreSQL 和 Feishu

## 执行要求

- 只能使用浏览器模拟流程。
- 不允许调用未授权接口。
- 每次任务必须保留发布前、发布中、发布后截图。
- 失败必须保留异常截图和错误日志。
- 支持人工审核状态控制。
