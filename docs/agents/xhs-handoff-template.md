# XHS Handoff Template

Use this template when closing a conversation window or handing a task to another agent.

## 1. Current status

```text
Task:
Branch:
Latest commit:
PR:
Status: not started / in progress / blocked / ready for review / merged
```

## 2. What changed

```text
Modified files:
- path/to/file
- path/to/file

Behavior added or changed:
- ...
```

## 3. Tests run

```text
Command:
Result:
```

## 4. Evidence / artifacts

```text
Evidence path:
Screenshot path:
JSON path:
Log path:
```

## 5. Decisions made

```text
- Decision:
  Reason:
  Source doc / issue / PR:
```

## 6. Known risks

```text
- Risk:
  Impact:
  Next action:
```

## 7. Next recommended task

```text
Task title:
Goal:
Allowed files:
Test command:
```

## 8. Do not forget

```text
- Do not submit local evidence directories.
- Do not submit local screenshots.
- Do not submit local profiles.
- Keep selenium_chrome debug-only.
- Keep real external calls out of unit tests.
```
