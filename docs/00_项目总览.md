# 00_项目总览

## 项目名称

xhs-automation

## 当前目标

搭建一个可被 Codex 在 VS Code 中稳定理解和开发的自动化项目框架。

第一阶段重点：
- 小红书热点追踪
- 小红书 RPA 发布与搜索
- 跨境卫士账号/店铺环境启动
- 影刀 RPA 页面 UI Flow 执行
- Feishu 多维表驱动
- n8n 工作流编排
- browser-worker 调度与 evidence 承接
- PostgreSQL 数据存储
- MinIO 截图与证据归档

## 总体架构

Feishu 多维表
→ n8n 定时或人工触发
→ browser-worker 接收 job
→ Provider Router 选择执行链路
→ 跨境卫士启动账号/店铺环境
→ 影刀 RPA 执行小红书页面 UI Flow
→ 输出 evidence JSON / 截图 / normalized_records
→ 结构化数据写入 PostgreSQL
→ 执行结果回写 Feishu

`selenium_chrome_provider` 仅保留为本地 debug，direct search URL 仅用于 debug，不作为主链路。

## MVP 原则

- 先跑通最小闭环。
- 先人机协作，不追求全自动。
- 所有关键动作必须可追踪。
- 所有失败必须保留截图证据。
- 所有自动化节点必须能人工接管。
