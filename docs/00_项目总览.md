# 00_项目总览

## 项目名称

xhs-automation

## 当前目标

搭建一个可被 Codex 在 VS Code 中稳定理解和开发的自动化项目框架。

第一阶段重点：
- 小红书热点追踪
- 小红书浏览器模拟发布
- Feishu 多维表驱动
- n8n 工作流编排
- browser-worker 执行
- PostgreSQL 数据存储
- MinIO 截图与证据归档

## 总体架构

Feishu 多维表
→ n8n 定时或人工触发
→ browser-worker 接收 job
→ 浏览器模拟执行
→ 截图 / 页面证据上传 MinIO
→ 结构化数据写入 PostgreSQL
→ 执行结果回写 Feishu

## MVP 原则

- 先跑通最小闭环。
- 先人机协作，不追求全自动。
- 所有关键动作必须可追踪。
- 所有失败必须保留截图证据。
- 所有自动化节点必须能人工接管。
