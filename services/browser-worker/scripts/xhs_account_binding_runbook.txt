XHS Account Binding Check Runbook

Purpose:
Confirm account_id, shop_id, and shop_name only against local profile map and readonly discovery evidence.

Hard boundaries:
- 不要 open shop.
- 不要 close shop.
- 不要打开小红书.
- 不要打开外部网页.
- 不要点击真实发布.
- 不要调用影刀云 API.
- Do not call Yingdao cloud API or OpenAPI.
- Do not call KuaJingVS open shop or close shop.
- Do not search or publish anything.

Manual confirmation steps:
1. Run xhs_account_binding_prepare.ps1 from PowerShell.
2. Open account_binding_context.json.
3. Open actual_form_fill_input.json.
4. Confirm account_id is the expected local account.
5. Confirm shop_id and shop_name match the local profile map and readonly discovery evidence.
6. Write account_binding_confirmation.json to the confirmation_path.
7. Keep all runtime real-action flags false:
   - opened_shop
   - closed_shop
   - opened_xhs
   - opened_external_url
   - called_yingdao_openapi
   - called_kuaijingvs_open_shop
   - real_search_executed
   - real_publish_executed
8. Return to PowerShell and run xhs_account_binding_verify.ps1.

This runbook only checks local JSON binding. It does not prove the shop environment was opened or that Xiaohongshu is logged in.
