Task 38 Local n8n/OpenClaw Contract Replay Runbook

Purpose
- Replay n8n and OpenClaw contracts only through browser-worker local mock routes.
- Carry strict account binding context and hardened discovery references in local payloads.
- Validate that replay payload/result/summary files contain no token, cookie, secret, password, auth, or header values.

Steps
1. Start browser-worker locally.
2. Run readiness.
3. Make sure hardened_discovery.json exists by running xhs_kjvs_discovery_harden.ps1 against local discovery evidence.
4. Run xhs_account_binding_strict_check.ps1 for the same job_id and account_id.
5. Run xhs_contract_replay_n8n_search.ps1 or xhs_contract_replay_n8n_publish.ps1.
6. Run xhs_contract_replay_openclaw_status.ps1.
7. Or run xhs_contract_replay_all.ps1 to perform strict binding plus both local replays.

Safety Boundary
- 不调用真实 n8n.
- 不调用真实 OpenClaw.
- 不调用影刀云 API.
- 不调用真实 Feishu/PostgreSQL/MinIO.
- 不要 open shop.
- 不要 close shop.
- 不要打开小红书.
- 不要打开外部网页.
- 不要点击真实发布.
- 不搜索小红书.
- 不发布小红书.
